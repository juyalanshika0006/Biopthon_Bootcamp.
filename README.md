# 🧬 Biopython Bootcamp

A hands-on learning repository documenting my journey of learning **Biopython** through practical coding exercises, Rosalind problems, and real-world bioinformatics workflows.

This repository serves as both a learning journal and a collection of reusable scripts for sequence analysis and biological data processing.

---

## 📌 About

The goal of this bootcamp is to develop strong programming skills in computational biology by solving practical problems using Python and Biopython.

Topics include:

- DNA & RNA sequence manipulation
- Protein translation
- FASTA file processing
- Sequence alignment
- NCBI database access
- BLAST searches
- Rosalind bioinformatics challenges
- Biological file handling
